# Classification-Model
This repo contains classification model comparison with the basic 2D dataset which includes some model like,
* Logistics Regression
* K-Nearest Neighbours
* Naive Bayes
* Support Vector Machines
* Kernel - SVM
* Decision Tree
* Bagging Classifier
* Random Forest
* Ada Boost
* Gradient Boost
* Light GBM
* XG Boost
* CAT Boost
